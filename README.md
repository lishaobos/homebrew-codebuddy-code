# Homebrew Tap for CodeBuddy Code

Official Homebrew tap for [CodeBuddy Code](https://cnb.cool/codebuddy/codebuddy-code) - AI-powered coding assistant for terminal, IDE, and GitHub.

Supports **macOS** and **Linux**.

## Install

```bash
brew tap lileo/codebuddy-code
brew install codebuddy-code
```

Or in a single command:

```bash
brew install lileo/codebuddy-code/codebuddy-code
```

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "lileo/codebuddy-code"
brew "codebuddy-code"
```

### Upgrade

```bash
brew upgrade codebuddy-code
```

### Uninstall

```bash
brew uninstall codebuddy-code
brew untap lileo/codebuddy-code
```
